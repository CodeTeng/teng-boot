# 后台管理系统使用手册

## 登录
1. 打开 http://localhost:5173
2. 输入管理员账号和密码
3. 拖动滑块验证码完成验证
4. 点击登录

## 控制台
- 查看用户总数、角色总数、菜单总数、文件总数
- 点击卡片可快速跳转到对应管理页面

## 用户管理
- 查看用户列表（分页、搜索）
- 新增用户（设置用户名、角色、手机号等信息）
- 编辑用户信息
- 修改用户密码
- 批量删除用户
- 导出用户数据为 Excel

## 角色管理
- 查看角色列表
- 新增/编辑/删除角色
- 为角色分配菜单权限（树形勾选）

## 菜单管理
- 查看菜单树形结构
- 新增目录/菜单/按钮
- 编辑/删除菜单
- 设置权限标识、路由地址、图标等

## 文件管理
- 查看已上传的文件列表
- 点击文件名预览图片
- 删除文件记录

## 操作日志
- 查看用户操作记录
- 按用户名、操作类型筛选
